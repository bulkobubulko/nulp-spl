# NULP-SPL

## Usage
1) Clone the repository:

   ```bash
   git clone https://github.com/bulkobubulko/nulp-spl
    ```
2) Go to the directory:

   ```bash
   cd nulp-spl
    ```

3) Install Dependencies:

   ```bash
   pip install -r requirements.txt
    ```

4) Run the Application:

   ```bash
   python src/runner.py
    ```
